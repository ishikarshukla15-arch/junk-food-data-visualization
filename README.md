#  Junk Food Data Analysis & Visualization

##  Project Overview

This project focuses on exploring and visualizing a junk food dataset using Python and Jupyter Notebook.

The analysis explores different aspects of junk food, including food types, brands, pricing, annual sales, popularity, and nutritional attributes. Various data analysis and visualization techniques were used to identify patterns and communicate insights from the dataset.

##  Objectives

- Explore and understand the junk food dataset
- Perform basic data analysis using Python
- Analyze relationships between different variables
- Compare food types, brands, prices, and sales
- Visualize data to identify patterns and trends
- Gain practical experience in exploratory data analysis and data visualization

##  Tools & Technologies

- Python
- Jupyter Notebook
- Pandas
- Matplotlib
- Seaborn

##  Visualizations Used

The project includes multiple types of data visualizations:

- Line Plot
- Bar Plot

- Area Plot

##  Analysis Performed

The project includes:

- Data exploration
- Descriptive statistics
- Data filtering
- Grouping and aggregation
- Comparison of food categories
- Brand-level analysis
- Price analysis
- Sales analysis
- Nutritional attribute analysis
- Popularity analysis

##  Dataset

The dataset contains information related to junk food products, including attributes such as:

- Food Type
- Brand
- Price
- Country
- Annual Sales
- Calories
- Fat
- Sugar
- Popularity Score

##  Key Insights

The visualizations help explore:

- Differences in annual sales across junk food types
- Price variations across different groups and countries
- Relationships between nutritional attributes such as fat and calories
- Contribution of leading brands to total annual sales
- Differences in popularity across food categories
- Patterns in pricing, sales, and nutritional characteristics
